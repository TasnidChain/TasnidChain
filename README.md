### Open Source

The **API Toll** ecosystem is fully open source — micropayment infrastructure for AI agents.

- [`APITOLL`](https://github.com/TasnidChain/APITOLL) — Core platform: 6 npm packages, 75+ endpoints, USDC on Base & Solana
- [`apitoll-demo`](https://github.com/TasnidChain/apitoll-demo) — Run an AI agent that pays for APIs in 30 seconds
- [`x402-gateway`](https://github.com/TasnidChain/x402-gateway) — x402 Payment Protocol Gateway

### Philosophy

I believe the Muslim world needs its own technical infrastructure — not borrowed, not retrofitted, but purpose-built. Every product I ship is designed around that thesis.

Knowledge should be verifiable. Finance should be halal. The internet should be safe. And builders should own what they create.
